Plan de relance priorise automatique

Classement par enjeu (solde a engager x retard de consommation). Le parametrage : un producteur apparait s'il a consomme moins de 80% de sa dotation ; il est marque prioritaire sous 50%. Commencez par le haut.

1. Import JSON (Cowork ou autre outil)

Envoyez vos factures (une ou plusieurs) a un agent Cowork (ou tout autre outil) avec le prompt fourni ci-dessous, recuperez le JSON en sortie, collez-le ici. Les lignes s'affichent ensuite pour verification : rien n'est enregistre automatiquement, vous validez chaque ligne (ou la corrigez) avant de l'enregistrer via l'encart 2.
Voir le prompt a donner a Cowork

2. Saisie manuelle (ou validation d'une ligne importee)

Utilisable seule pour une saisie 100% manuelle, ou pre-remplie automatiquement en cliquant "Utiliser" sur une ligne de l'encart 1. Tous les champs du format officiel FranceAgrimer sont couverts ici. Rien n'est enregistre tant que vous ne cliquez pas sur "Enregistrer cette ligne".

3. Import questionnaire producteurs (Excel Jotform) - depenses declarees

Importe directement le fichier Excel exporte depuis Jotform (une ligne par producteur, toutes les mesures en colonnes). Chaque depense renseignee devient une ligne avec Origine = "declaree" - a confirmer individuellement (bouton dans l'onglet Depenses) une fois la vraie facture recue.
